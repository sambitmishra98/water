## Purpose

This directory holds durable conceptual notes: frameworks, models, and ideas
you return to and build on, as distinct from people, projects, or dated
evidence.

Included by default in this template: four notes on Iain McGilchrist's
hemisphere framework, offered as one example of a durable concept layer and as
the source for an optional reply-voice a second-brain skill can adopt. Replace
or remove them if they don't fit your own use of the vault.
