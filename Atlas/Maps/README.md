## Purpose

This directory holds maps: notes that explain how to think through a part of
the vault, as distinct from notes that hold the knowledge itself.

Included by default: `Second Brain Operating Model.md`, which explains the
WATER layers as AI-readiness levels and documents one way to adapt Andrej
Karpathy's llm-wiki pattern (a persistent, LLM-maintained wiki instead of pure
RAG) under a propose-don't-resolve discipline. Treat it as a worked example,
not a fixed requirement.
