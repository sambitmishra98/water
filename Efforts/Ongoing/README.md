## Purpose

This directory holds active, ongoing efforts that aren't tied to one profession
folder: standing practices, running experiments, things you keep coming back
to and refining.

Included by default: `The right-left-right approach.md`, documenting the
daily triad of `/morning-descent`, `/session`, and `/evening-reflect` as one
worked example of a standing practice. Replace or remove it if it doesn't fit
how you want to work.
