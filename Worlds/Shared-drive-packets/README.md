## Purpose

`Worlds/Shared-drive-packets` stores access-controlled packet manifests.

Google Drive is valid for controlled sharing.

## Rules

- Store manifest notes and access notes, not raw archives.
- Do not create topic folders before a first packet exists.
- God or alternate-currency work belongs here only when the first Drive packet is ready, and not in `Worlds/Online-presence` by default.
- Each packet should include audience, purpose, access method, link or placeholder, included files, source notes, privacy status, and review status.
