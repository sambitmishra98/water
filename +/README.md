## Purpose

`+` is the inbox: raw, unsorted capture. Anything unstable, half-formed, or
not yet placed lands here first and is later triaged into the WATER layers
(see the root `README.md`).

## Rules

- Capture freely; do not demand structure at capture time.
- Triage is a deliberate act: promote to Atlas/Efforts/Research/Timeline/
  Worlds when a note stabilizes, then delete or empty the stub here.
- An LLM should treat inbox notes as pre-triage staging: propose promotions,
  never bulk-delete (some stubs are seeds).
