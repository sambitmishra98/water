## Purpose

`Research/profession/ExampleProject` holds source mapping, proposal context,
research questions, and uncertain synthesis for this project.

## Boundaries

- Externally received code, results, or context are received input, kept
  distinct from your own work on it.
- Active attempts to run or modify code belong in
  [[Efforts/profession/ExampleProject/README]].
- Stable concepts later distill into [[Atlas/Profession/ExampleProject/README]].
- Shareable summaries later move to
  [[Worlds/ResearchGroups/ExampleGroup/Projects/ExampleProject/README]].
- Heavy archives stay in `Supplementary/on-drive`, Drive, or another external
  archive.
